# 汉字分析工具
<h2>功能</h2>
<h4>-查看超过一万七千个*汉字的部首组成</h4>
<h4>-搜索指定汉字的部首组成</h4>
<h4>-搜索包含指定部首的汉字</h4>
<h4>-统计部首在汉字中出现的次数</h4>
<h4>-导出视图到文件</h4>
<i>*: 取决于数据源</i><br>
<h2>特性</h2>
<h4>-多语言支持</h4>
 -支持自动使用操作系统语言<br>
 -支持运行时变更语言
<h4>-高 DPI 支持</h4>
<h4>-超级快</h4>
